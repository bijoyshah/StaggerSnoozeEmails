# Gmail Stagger Snooze 💤

Stop drowning in your inbox. This Chrome extension snoozes selected Gmail emails in staggered intervals so they come back to you one at a time. Deal with each email when you're ready, reduce context switching, and actually get things done.

---

## Download & Install

### Step 1 — Download
Click the green **Code** button at the top of this page → **Download ZIP**

Unzip the downloaded file on your computer.

### Step 2 — Open Chrome Extensions
Type this in your Chrome address bar and hit Enter:
```
chrome://extensions/
```

### Step 3 — Enable Developer Mode
Toggle on **Developer mode** in the top right corner of the page (it turns blue when on).

### Step 4 — Load the Extension
Click **Load unpacked** → select the unzipped folder → click **Open**

### Step 5 — Go to Gmail
Go to [mail.google.com](https://mail.google.com) and look for the 💤 button in the bottom right corner. Click it to get started!

---

## How to Use

1. Check emails in the order you want them snoozed
2. Click the 💤 button in the bottom right corner of Gmail
3. Set your start time and increment
4. Click **Snooze Selected**

Click the ❓ inside the extension panel for more tips.

---

## Features

- Snooze multiple emails in staggered time increments
- Select emails in priority order — first clicked = first to come back
- Set a custom start time (auto-defaults to the next 15-minute mark)
- Choose your increment: 5, 15, 30 minutes or 1 hour
- Remembers your last used increment
- Collapsible sidebar — stays out of your way when not in use
- No login, no account, no data collection

---

## Troubleshooting

- **Don't see the 💤 button?** Refresh Gmail after installing
- **Something broken?** Go to `chrome://extensions/`, find Gmail Stagger Snooze, click the refresh icon, then reload Gmail
